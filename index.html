About Me

Halo! Saya adalah seorang Junior Content Creator, Junior Web Developer, dan Junior Mobile App Developer yang memiliki semangat tinggi dalam dunia digital dan teknologi.

Sebagai Content Creator, saya memiliki pengalaman dalam membuat konten kreatif yang menarik, baik dalam bentuk tulisan, gambar, maupun video. Saya selalu berusaha menyajikan ide-ide segar dan git inovatif agar dapat memberikan nilai tambah bagi audiens.

Dalam bidang Web Development, saya memiliki kemampuan dasar dalam membangun dan mengelola website menggunakan HTML, CSS, JavaScript, serta beberapa framework seperti Laravel. Saya senang mengeksplorasi teknologi terbaru untuk menciptakan pengalaman pengguna yang menarik dan responsif.

Sebagai Junior Mobile App Developer, saya juga memiliki pemahaman dalam pengembangan aplikasi mobile, khususnya menggunakan Flutter Saya antusias dalam menciptakan aplikasi yang tidak hanya fungsional, tetapi juga user-friendly. Saya selalu terbuka untuk belajar dan berkembang, serta siap berkolaborasi dalam berbagai proyek kreatif dan teknologi. Jika Anda tertarik untuk bekerja sama, jangan ragu untuk menghubungi saya!
